# under_line
### Purpose
Display a line of a single character under a line of text.
### Arguments
$1 -> A single line of text to be displayed.
$2 -> Optional character to form the line. Default is `-`line.
### Returns
Displays a line of text underlined by a string of a single character.
### Usage
```bash
```
### Notes
- The line of text cannot contain any special characters.
- The character argument, if present, needs to be in double quotes, i.e., "-".
- A hyphen is the default character if no character is specified.
- If more than one character appears in the string, only the first character is used.
