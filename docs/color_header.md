# color_header
### Purpose
Display a string of text centered on a colored background
### Arguments
$1 -> A string of text to be displayed
$2 -> An optional background color.
### Usage
```bash
color_header "Text string" [COLOR]
```
### Notes
- Red is the default if no color is provided.
- The color argument is case insensitive.
- The colors are aqua, black, blue, green, gray (or grey), orange, purple, and white.
