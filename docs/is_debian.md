# is_debian
### Purpose
Check if the distribution is based on Debian,.
### Arguments
None
### Returns
Returns 0 (TRUE) if Debian, LMDE, Antix, or MX Linux, otherwise returns 1 (FALSE).
### Usage
```bash
is_debian && do someting
```
### Notes
Antix, BunsenLabs, and MX Linux show as a Debian codename.
